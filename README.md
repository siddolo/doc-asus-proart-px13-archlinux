# ASUS ProArt PX13 Arch Linux

Note e procedure per la configurazione di un portatile ASUS ProArt PX13 (HN7306EAC) con Arch Linux.
Il PC è basato su AMD Ryzen AI Max+ 395, piattaforma Strix Halo GPU integrata Radeon 8060S (ROCm) e 128GB di memoria RAM condivisa.
L’hardware è interessante soprattutto per workload AI locali vista la quantità di RAM.

Alcune parti della documentazione sono specifiche dell’ASUS ProArt PX13 (audio, touchscreen, gestione energetica, sospensione, ibernazione e workaround legati al firmware del portatile). Altre sezioni, invece, possono essere adattate con poche modifiche a piattaforme Strix Halo/ROCm simili, come GMKtec NucBox EVO-X2, Framework Desktop, e simili.
